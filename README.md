# Image To Webp

> Image To Webp, çalıştığı klasordeki tüm resimleri webp formatına dönüştürmenizi sağlayan bir Python projesidir.

<br> <br>

## SETUP 

### setup - 1

``` batch
uv venv
uv pip install ./dist/image_to_webp-1.0.0-py3-none-any.whl
```

<br>

### setup - 2
``` batch
pip install -U -r requirements.txt
```


<br><br><br>

## RUN

### run - 1 (with uv)

```batch
uv run main.py
```

<br>

### run - 2 (python)

``` batch
python.exe main.py
```

<br>

### run - 3 (exe)

> click to run.exe  or 
```batch
call run.exe
```
