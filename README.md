# Image To Webp

# SETUP 

## setup - 1

``` batch
uv venv
uv pip install ./dist/image_to_webp-1.0.0-py3-none-any.whl
```

## setup - 2
``` batch
pip install -U -r requirements.txt
```

# RUN

## run - 1 (with uv)

```batch
uv run main.py
```

## run - 2 (python)

``` batch
python.exe main.py
```

## run - 3 (exe)

> click to run.exe  or 
```batch
call run.exe
```
